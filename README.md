# Blockchain Project

Një implementim i thjeshtë i blockchain-it në Python.

## Struktura e Projektit
- `blockchain.py`: Klasat Block dhe Blockchain.
- `consensus.py`: Mekanizmi i konsensusit Proof of Work.
- `main.py`: Shembull përdorimi.
